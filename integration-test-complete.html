How to Use

To generate embed codes for Teachable lessons:

  1. Open teachable-embed-generator-clean.html in your browser
  2. Fill in the course level and lesson number
  3. Keep the default GitHub Pages integration URL
  4. Add your audio and VTT file URLs (they can be hosted on Dropbox)
  5. Click "Generate Code" and copy the generated code into your Teachable lesson

Embed Code Example

<!-- Dictation Exercise - Course A1, Lesson L01 -->
<div class="dictation-app-container" 
     data-course-id="A1" 
     data-lesson-id="L01" 
     data-audio-url="https://dl.dropboxusercontent.com/s/your-file-path/audio.mp3?raw=1"
     data-vtt-url="https://dl.dropboxusercontent.com/s/your-file-path/audio.vtt?raw=1">
</div>

<!-- Teachable Integration Script -->
<script src="https://smarter-german-org.github.io/NEW-DICTATION-CHECKER/teachable-integration.js"></script>

Troubleshooting

If you encounter issues with the integration:

  1. Make sure your audio and VTT files are properly shared (with raw=1 parameter if using Dropbox)
  2. Check browser console for any error messages
  3. Verify that GitHub Pages deployment is working by visiting https://smarter-german-org.github.io/NEW-DICTATION-CHECKER/
  4. If needed, refer to the troubleshooting guide
